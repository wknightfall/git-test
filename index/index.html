git init (初始化git)
git config (配置git user.name user.email)
git status (查看工作区，暂存区状态)
git add file_name (把工作区的代码提交到暂存区)
git commit -m "remark"(把暂存区的代码提交到本地库)
git log (完整日志 空格向下翻页，b向上翻页，q退出)
git log --pretty--oneline(单行显示日志信息 简洁)
git log --oneline (缩短哈希值显示，更简洁)
git reflog (显示到某个版本需要移动几步)
git reset --hard 哈希值 (跳转到当前哈希值的版本)（配合 git reflog）
git reset --hard HEAD^ (只能后退版本，几个“^”退几步) （配合 git log --oneline）
git reset --hard HEAD~number (只能后退版本，number是多少就退多少步)（配合 git log --oneline）
--soft (移动本地库HEAD指针) --mixed (移动本地库 HEAD指针,重置暂存区) --hard (移动本地库HEAD指针，充值暂存区，重置工作区)
